# 摸鱼小助手 - 产品流程图

## 整体架构

```
┌──────────────────────────────────────────────────────┐
│                    main.py (入口)                     │
│  ├── --mobile → mobile/app.py (Kivy)                │
│  └── 默认    → desktop/app.py (Tkinter+ttkbootstrap) │
└──────────────────────────────────────────────────────┘
                         │
           ┌─────────────┼─────────────┐
           ▼             ▼             ▼
    ┌────────────┐ ┌───────────┐ ┌──────────────┐
    │   core/    │ │  desktop/ │ │   mobile/    │
    │  共享逻辑  │ │  桌面 UI   │ │  移动 UI     │
    └────────────┘ └───────────┘ └──────────────┘
```

## 核心模块关系

```
core/
 ├── reminder_types.py      # 提醒类型定义 + 消息模板
 │    └── ReminderType (枚举): drink_water, kegel, fish_touch, rest_eyes, stretch
 │
 ├── reminder_scheduler.py  # 定时调度器
 │    └── ReminderScheduler  ──→ 回调 UI 弹窗
 │
 ├── process_monitor.py     # 进程监控器
 │    └── ProcessMonitor  ──→ psutil 扫描进程
 │
 ├── settings_manager.py    # 设置持久化 (JSON)
 │    └── SettingsManager  ──→ ~/.fish_assistant/config.json
 │
 ├── image_manager.py       # 表情包管理器
 │    └── FunnyImageManager ──→ ~/.fish_assistant/images/{type}/
 │
 └── ui_helpers.py          # UI 工具函数 (颜色/进度条/激励语)
```

## 定时调度循环 (每10秒)

```
┌──────────────────────┐
│  ReminderScheduler   │
│  _loop() 后台线程     │
└──────┬───────────────┘
       │ 每 10 秒
       ▼
┌──────────────────────┐     ┌──────────────────────────────┐
│ _check_daily_reset() │────▶│ 过了午夜？                    │
│ 检查是否需要日重置    │     │ 是 → 重置统计/提醒计数        │
└──────┬───────────────┘     └──────────────────────────────┘
       │
       ▼
┌──────────────────────┐     ┌──────────────────────────────┐
│ _monitor.update()    │────▶│ psutil.process_iter(['name'])│
│ 更新进程监控数据      │     │ 扫描所有运行中的进程          │
│                      │     │ 匹配 → 累加使用时间 (+10s)   │
└──────┬───────────────┘     └──────────────────────────────┘
       │
       ▼
┌──────────────────────┐     ┌──────────────────────────────┐
│ _check_app_limits()  │────▶│ 某APP使用超过限制时间？        │
│ 检查APP使用时间限制   │     │ 是 → 触发摸鱼警告提醒         │
└──────┬───────────────┘     │ (每个APP只提醒一次/天)        │
       │                     └──────────────────────────────┘
       ▼
┌──────────────────────┐     ┌──────────────────────────────┐
│ _check_periodic()    │────▶│ 到了提醒间隔？                │
│ 检查周期提醒          │     │ 是 → 随机选一种已启用的提醒   │
│                      │     │      触发 _fire_reminder()   │
└──────────────────────┘     └──────────────────────────────┘
```

## 提醒弹窗流程

```
_fire_reminder(type, msg)
       │
       ▼
┌──────────────────────┐
│ 回调 → UI线程        │
│ _on_reminder()       │
└──────┬───────────────┘
       │
       ▼
┌──────────────────────┐
│ _show_reminder_popup │
│ 创建 Toplevel 弹窗   │
└──────┬───────────────┘
       │
       ├──→ 标题 + Emoji + 提醒类型
       ├──→ ASCII Art 搞笑文字
       ├──→ 随机提醒消息

       └──→ "知道了" 按钮 → 关闭弹窗
```

## 表情包图片管理

```
~/.fish_assistant/images/
  ├── drink_water/     💧 喝水提醒
  │   ├── meme_00123.png    (程序生成)
  │   ├── meme_00456.png    (程序生成)
  │   └── my_custom.jpg     (用户自定义)
  ├── kegel/           🍑 提肛提醒
  ├── fish_touch/      🐟 摸鱼提醒
  ├── rest_eyes/       👀 护眼提醒
  └── stretch/         🧘 伸展提醒

获取图片流程:
  get_random_meme(type)
    │
    ├─→ 子目录有图片？ ──→ 随机选一张返回
    │
    └─→ 没有图片？ ──→ generate_meme_image()
                        │
                        ├─→ 随机选模板 (emoji + top_text + bottom_text)
                        ├─→ 随机选背景色
                        ├─→ Pillow 渲染 400x400 图片
                        └─→ 保存到子目录 → 返回路径
```

## Windows 进程监控原理

```
ProcessMonitor.update()
       │
       ▼
┌───────────────────────────────────────┐
│ psutil.process_iter(['name'])         │
│                                       │
│ Windows 底层:                         │
│   → 调用 NtQuerySystemInformation()  │
│   → 或 EnumProcesses() / OpenProcess │
│   → 遍历系统所有进程，获取进程名       │
│   → 类似"任务管理器"看到的进程列表     │
└───────────────┬───────────────────────┘
                │
                ▼
┌───────────────────────────────────────┐
│ 匹配 MONITORED_APPS 字典              │
│                                       │
│  "wechat":  ["WeChat.exe"]           │
│  "douyin":  ["Douyin.exe",           │
│              "DouYinPlayer.exe"]      │
│  "bilibili":["Bilibili.exe"]         │
│  "qq":      ["QQ.exe", "QQMusic.exe"]│
│                                       │
│ 进程名匹配（不区分大小写）？           │
│  ├── 匹配 → usage.total_seconds += 10│
│  └── 不匹配 → 跳过                    │
└───────────────────────────────────────┘
```

## 技术栈总览

| 层级 | 技术 | 用途 |
|------|------|------|
| UI (桌面) | Tkinter + ttkbootstrap | 跨平台窗口/主题 |
| UI (移动) | Kivy | Android/iOS |
| 进程监控 | psutil | 跨平台进程扫描 |
| 图片生成 | Pillow | 表情包渲染 |
| 持久化 | JSON (stdlib) | 设置存储 |
| 打包 | PyInstaller | Windows EXE |
| 测试 | pytest | 143 单元测试 |
